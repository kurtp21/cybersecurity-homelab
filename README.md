# 🛡️ Cybersecurity Homelab (100 Days Challenge)

## 📌 Overview

This repository documents my 100-day journey building a cybersecurity homelab focused on blue team operations.

## 🎯 Goals

* Build a virtual enterprise lab (Active Directory, SIEM, endpoints)
* Practice detection engineering and incident response
* Simulate real-world cyber attacks and analyze them
* Prepare for roles in:

  * SOC Analyst
  * Cybersecurity Analyst
  * Network / Cloud Security Engineer

## 🧰 Tools & Technologies

* VirtualBox
* Windows Server
* Linux (Ubuntu)
* SIEM (Splunk / Wazuh)
* Sysmon
* Wireshark

## 🗂️ Repository Structure

* `/setup` → Installation & configuration logs
* `/network-diagrams` → Lab architecture
* `/detection-rules` → SIEM rules & queries
* `/incident-reports` → Security investigations
* `/screenshots` → Visual proof of work
* `/notes` → Concepts & learnings
